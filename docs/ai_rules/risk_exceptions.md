# 风险例外规则

## 必须优先提示的风险

- 系统性风险
- 板块轮动风险
- 持仓集中风险
- 政策、监管、地缘、海外市场扰动

## 例外处理

- 指数、成交量、板块、快讯信号互相矛盾时，必须明确写“信号分化”。
- 如果数据不充分，必须承认数据不足，不能强行补判断。
- 如果外部新闻来自搜索结果，必须保留来源意识，不能把它表述成绝对事实。
- 如果搜索结果与页面实时数据冲突，必须点明冲突来源和不确定性。

## 深度研究附加要求

- 每次只调用一个工具后再决策下一步，避免无控制扩张。
- 对于重大事件，必须遵循“背景 -> 影响 -> 启示”的解释顺序。
- 结尾反复保留“仅供参考，不构成投资建议”的风险提示。
