# 输出格式规则

## 通用格式

- 先给证据，再给判断，最后给建议。
- 使用结构化 Markdown，避免大段无分层文本。
- 不能输出确定性收益承诺、确定性买卖结论、绝对化措辞。

## 标准分析适用

- 输出应覆盖市场趋势、板块机会、基金建议、风险提示四部分。
- 每一部分都要有“现象 + 判断 + 建议/提示”。

## 快速分析适用

- 仅输出四段：市场趋势、板块机会、基金建议、风险提示。
- 每段都必须包含一句风险或约束，不可只写乐观观点。

## 深度研究适用

- 报告必须包含：市场趋势、板块机会、基金组合建议、风险提示、信息来源说明。
- 网络搜索获得的信息应尽量保留 Markdown 链接引用。
- 持仓基金分析必须结合基金表现与相关新闻共同展开。
