# 入场规则

## 通用入场条件

- 先看快讯方向，再看指数，再看成交量，最后看板块扩散。
- 指数、成交量、板块三项中至少两项共振，才允许输出偏积极判断。
- 若快讯偏利多但指数和成交量不配合，只能表述为“观察改善信号”，不能直接给偏强结论。
- 若领涨板块只有单一主题、缺少扩散，必须提示“主题驱动但持续性待验证”。

## 标准分析适用

- 当板块强、成交量放大、指数同步走强时，可表述“风险偏好修复”或“阶段性回暖”。
- 只对数据中实际出现强势表现的板块展开，不推演未出现的热点。

## 快速分析适用

- 只保留最强入场信号，不展开冗长论证。
- 数据不足时直接提示“样本有限，仅供观察”，不要凑完整看多逻辑。

## 深度研究适用

- 深度模式可以围绕重要快讯、强势板块、持仓相关主题继续搜索原文。
- 搜索扩展必须服务于已有信号，不允许无边界发散。
