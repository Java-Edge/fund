# 仓位控制规则

## 通用仓位原则

- 先评估持仓匹配度，再讨论增减仓，不要跳过仓位结构直接谈方向。
- 如果持仓集中在单一主题、单一风格或单一风险因子，必须提示集中度风险。
- 如果市场信号分化，建议优先表述“控制仓位、分批观察、保留灵活性”。

## 标准分析适用

- 调仓建议只能使用“偏增持/偏减持/继续持有/继续观察”这类非确定性措辞。
- 当市场环境与持仓风格不匹配时，应先提出仓位优化方向，再谈具体板块。

## 快速分析适用

- 仓位建议压缩为一句，但不能省略。
- 优先表达“保持中性”“避免满仓式判断”“控制单主题暴露”。

## 深度研究适用

- 应结合基金表现、相关新闻、行业动态共同判断仓位暴露。
- 深度报告里要区分短期仓位应对和中期配置方向。
