# 减仓规则

## 规则名称：共振走弱时提高减仓或防守权重

- 触发条件：需要给出偏防守、偏减仓、降低暴露的判断时。
- 证据要求：指数走弱、成交量萎缩、强势板块退潮三项中出现两项以上；或持仓基金上涨但支撑板块明显转弱。
- 输出限制：建议改为“逐步收缩风险暴露”而不是继续追涨。
- 风险提示：若快讯出现监管、政策收紧、海外冲击等负面扰动，结论必须降一级。

## 规则名称：标准分析中的减仓措辞限制

- 触发条件：标准分析模式下输出减仓建议时。
- 证据要求：需要有市场走弱、板块退潮或持仓集中等可说明的依据。
- 输出限制：建议只能写成“偏减持/降低暴露/继续观察”，不能写确定性卖出指令。
- 风险提示：对单一主题高集中持仓，应优先提示分散化，而不是只说止盈。

## 规则名称：深度研究中的减仓理由链

- 触发条件：深度研究模式中，外部新闻和页面数据都指向风险抬升时。
- 证据要求：新闻和页面数据共同指向基本面或情绪恶化。
- 输出限制：如果外部搜索和页面数据冲突，先提示冲突，不直接给强减仓结论。
- 风险提示：必须把减仓理由拆成证据链，而不是一句话跳结论。
