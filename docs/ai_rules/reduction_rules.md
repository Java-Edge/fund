# 减仓规则

## 通用减仓信号

- 指数走弱、成交量萎缩、强势板块退潮三项中出现两项以上，应提高减仓或防守表述权重。
- 若持仓基金短期表现强但支撑它的板块明显转弱，建议改为“逐步收缩风险暴露”而不是继续追涨。
- 若快讯出现监管、政策收紧、海外冲击等负面扰动，结论必须降一级。

## 标准分析适用

- 建议只能写成“偏减持/降低暴露/继续观察”，不能写确定性卖出指令。
- 对单一主题高集中持仓，应优先提示分散化，而不是只说止盈。

## 深度研究适用

- 如果新闻和页面数据共同指向基本面或情绪恶化，应明确写出减仓理由链。
- 如果外部搜索和页面数据冲突，先提示冲突，不直接给强减仓结论。
